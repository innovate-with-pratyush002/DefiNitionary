# DefiNitionary

DefiNitionary is a small dictionary website made with HTML, CSS, and JavaScript.  
You can type any English word and it will show the meaning, example sentence, pronunciation, synonyms, and antonyms.

Live link: https://innovate-with-pratyush002.github.io/DefiNitionary/

## What this project can do

- Search any English word
- Show word meanings
- Show pronunciation text and audio
- Show synonyms and antonyms
- Save recent searches in the browser
- Change theme between dark and light

## Tools used

- HTML
- CSS
- JavaScript
- Free Dictionary API

## Why I made this

I made this project to practice JavaScript, API fetching, DOM manipulation, and UI design in a simple way.

## API used

This project uses the Free Dictionary API: https://dictionaryapi.dev/
